객체(Object)

  • 상요자 정의 객체 : 사용자에 의해서 만들어지는 객체
  • 데이터형 객체 : 데이터를 저장하고 처리하기 위한 내장 객체
  • BOM 객체 : 브라우저를 조작하기 위한 내장 객체
  • DOM 객체 : 문서 요소(Element)를 처리하기 위한 내장 객체

모든 객체는 메서드와 속성으로 나누어 진다.

객체의 메서드는 정의할 때는 var객체명.메서드명 = function() { 처리문장; }

객체의 메서드를 호출할 때는 객체명.메서드명()

객체 중에서 내장객체는 메서드나 속성이 정해져 있다.

객체의 속성을 정의할 때는 var 객체명.속성 = 값;

객체의 속성을 이용할 때는 객체명.속성

ex) 김기태는 성격도 좋고, 나이도 젊고, 잘생겼으며, 자주 영화를 보러 간다.

객체.성명 = "김기태"

객체.성격 = "좋다"

객체.나이 = "젊다"

객체.생김새 = "잘생김"

객체.행위 = function(){ 영화를 보러간다; }
